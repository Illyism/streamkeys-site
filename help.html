Install

Installing is simple! You can add the extension to Chrome from the Chrome Webstore

Configuration

All hotkeys can be changed. Simply go to the chrome extensions tab (get there from Chrome menu -> tools -> extensions). At the bottom there is a button for "Keboard Shortcuts" where you can change the hotkeys to those of your choosing. In this menu you can also set the hotkeys to be global across either Chrome or your system (see Global Hotkeys section for more info).

Default Hotkeys

Mac:

Play/Pause: Ctrl + Shift + 2
Next Song: Ctrl + Shift + 3
Previous Song: Ctrl + Shift + 1
Mute: Ctrl + Shift + M

Windows:

Play/Pause: Media Key Play
Next Song: Media Key Next
Previous Song: Media Key Previous
Mute: Alt + Shift + M

Global Hotkeys

By default, all hotkeys will work across the chrome browser. As of Chrome Beta 35 (verified on v35.0.1916.27 and above), extension hotkeys will now work globally across your whole computer. To get Streamkeys working from anywhere install Google Chrome Beta and restart your browser. The hotkeys should be enabled globally by default, if that doesn't work check the hotkey settings using the steps outlined in the Configuration section
