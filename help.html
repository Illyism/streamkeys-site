Install

Installing is simple! You can add the extension to Chrome from the Chrome Webstore

Configuration

All hotkeys can be changed. Simply go to the chrome extensions tab (get there from Chrome menu -> tools -> extensions). At the bottom there is a button for "Keboard Shortcuts" where you can change the hotkeys to those of your choosing

Mac users: Currently there is an issue with capturing media key presses in chrome on macs. This issue is slated to be fixed in the next Chrome release, but until then you will have to use other keys, sorry!

Default Hotkeys

Mac:

Play/Pause: Ctrl + Shift + 2
Next Song: Ctrl + Shift + 3
Previous Song: Ctrl + Shift + 1
Mute: Ctrl + Shift + M

Windows:

Play/Pause: Media Key Play
Next Song: Media Key Next
Previous Song: Media Key Previous
Mute: Alt + Shift + M

Global Hotkeys

By default, all hotkeys will work across the chrome browser. Get beta to make them work globally on your machine. Beta link.
